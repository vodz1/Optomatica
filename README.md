
**Back-End Installation:**

1. **Clone the back-end repo:**
   ```
   git clone https://github.com/vodz1/Optomatica.git
   ```
2. **Navigate to the project directory:**
   ```
   cd Backend
   ```
3. **Install dependencies using npm:**
   ```
   npm install
   ```
7. **Run the NestJS Backend:**
   ```
   npm run start:dev
   ```
